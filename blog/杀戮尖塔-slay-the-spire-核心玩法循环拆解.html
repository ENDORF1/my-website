# 玩法拆解

杀戮尖塔（Slay the Spire）核心玩法循环拆解

2026-04-03 Roguelike，独立游戏，DeckBuilding
写在最前：今天想给大家分享的是我最最喜爱的卡组构建-Roguelike游戏，也就是早已鼎鼎大名的杀戮尖塔。我在电脑端上有着差不多快400小时，手机端上有差不多100小时的游戏经历，我想这足以证明我对它的喜爱。

一、核心循环总览：三层嵌套结构

杀戮尖塔的玩法循环由三个层次构成，彼此嵌套、互相强化。理解这三层结构是分析整个游戏设计的基础。

 

1.1  宏观循环

时长：约 45–90 分钟 / 局

 

开始新局

→

选择角色

→

地图探索

→

Boss 挑战

→

通关 / 死亡

→

解锁内容

→

再次挑战

驱动力：角色解锁 → 不同卡池 / 遗物 → 更高难度（Ascension A1–A20）→ 再次挑战

 

宏观循环的核心驱动是"解锁-成长"的元进度机制。每次通关后解锁新的 Ascension 难度级别，也就是所谓的进阶难度，为核心玩家提供持续挑战阶梯。每个角色有独立卡池和专属遗物，驱动玩家体验不同流派。

 

1.2  中观循环（Meso Loop）

时长：约 5–15 分钟 / 楼层

 

查看地图

→

→

下一节点

驱动力：卡组状态 → 路径风险/收益判断 → 当前构筑目标

 

中观循环是战略层，玩家在此评估资源状态与风险，决定走精英（高收益/高风险）还是走安全路线。地图的有限可见性（只能看到前 2–3 步）制造持续的战略压力。

 

1.3  微观循环（Micro Loop）

时长：约 1–5 分钟 / 场战斗

敌方行动

驱动力：能量限制（3 点）→ 手牌优化 → 意图预判 → 伤害 / 防御平衡

 

微观循环是战术层，每回合 3 点能量的硬性约束制造出牌的取舍压力，而完全透明的敌方意图（Intent）系统将战斗从"随机博弈"转化为"资源规划谜题"。
